**SYNTAX:**

**~**draw `<type>` `<id>` `[COLORS]`

`type` - ball, ship, icon, spider, robot, wave, ufo, mini

`id` - 1,2,3... refer to http://geometry-dash.wikia.com/wiki/Icon_Kit

`[COLORS]` - Can be 6 RGB values. `R1 G1 B1 R2 G2 B2`
	   - Can be 2 HEX values. `#ff00ff #ff0bb`
	   - Can be 2 LITERAL values.
```
red, green, blue, yellow, orange, lime, cyan,
purple, magenta, pink, white, black, gray, denim, violet,
aquamarine, lightblue, darkblue, darkmagenta, lightpink,
lightorange, lightgray, emerald, darkcyan
```

**OR** 

**~**draw `random` `[TYPE]`

Not providing a type in the `random` function will just draw a random icon. Providing a type will draw a random icon of that type.

If no colors are provided icon will either be colored in random colors or black.
